# YA Pathfinder

Repository containg the Drupal files for the Young Adult (YA) literature pathfinder website I worked on for a library science class I took in Summer 2021. 

## Installation

To run this website on your local server, download the <i>pathfinder</i> folder from this repository and save it inside the <i>htdocs</i> or another folder for your web applications (depending on your OS) of your local server application (MAMP, XAMPP, WAMP). Then, start the server and go to your browser and type http://localhost:8888/pathfinder/, to access the Pathfinder.
